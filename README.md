## Descripción Breve
En este trabajo creamos y documentamos distintas funciones para el núcleo lógico de un sistema embebido que controlará semáforos en 
intersecciones clave de la ciudad, la implementación es en Common Lisp, también usamos local-time de Quicklisp e implementamos 2
funciones en Scala. Además, en el repositorio se pueden encontrar un informe, un código de honor respecto al uso de IA y un link al video
de YouTube donde explicamos la implementación de las funciones y mostramos sus casos de prueba.
